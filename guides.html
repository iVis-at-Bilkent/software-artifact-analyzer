Documentation

Software Artifact Analyzer (SAA) may be used as is, using the sample deployment provided or a local deployment of your own (as described here) without any customization. A User Guide was prepared to help when using SAA.

If you are a researcher or a practitioner who would like to implement your own analytics methods and integrate into SAA, you should start by forking this repo. The details of how SAA can be customized can be found in a Developer Guide we have prepared for you.
